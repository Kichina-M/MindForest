# MindForest
Це веб-платформа для підлітків, що допомагає їм краще розуміти власні емоції, знижувати рівень стресу та розвивати навички саморегуляції.
Підлітки часто не розуміють свої емоції та не мають простого інструменту для саморегуляції. Багато хто не розуміє що саме відчуває, через що починає ще більше хвилюватися.
Ціль першої ітерації: створити початковий набір з 5 вправ для зниження тривожності та приклад можливих проявів емоцій.
Основна цінність продукту - дати підлітку відчуття розуміння своїх емоцій та інструмент для самозаспокоєння.
Підлітки, які відчувають тривожність, психологи, а також розробник продукту.
Приблизно 10 користувачів написали відгук про покращення свого стану після використання пробної версії, кількість відвідувачів веб-платформи досягла хоча б 30.
