# MindForest
# Пітч
Це веб-платформа для підлітків, що допомагає їм краще розуміти власні емоції, знижувати рівень стресу та розвивати навички саморегуляції. Користувач використовує дану веб-платформу із бажанням позбутися постійного неспокою, навчитися розуміти та контролювати свої ємоції. Через деякий час постійного виповнення запропонованих завдань людина краще розуміє саму себе і може краще себе конролювати.
# Проблеми та опис сценарію
Підлітки часто не розуміють свої емоції та не мають простого інструменту для саморегуляції. Як спосіб заспокоїти себе чи детально вивчити свої емоції користувач відвідує веб-дизайн. На початку йде реєстрація, а саме ім'я користувача, вік та пошта. Після реєстрації користувач потрапляє у невеликий ліс, де й буде відбуватися не тільки спостереження за природою, але й виповнення завдань. Через деякий час використання веб-платформи людина відчуває легкість, більшу собраність та впевненість в собі.
# Ціль першої інтерації
Створити початковий набір з декількох легких вправ для зниження тривожності та приклад можливих проявів емоцій. Також додається пробна візуалізація віртуального лісу, де в ролі медитації можна спостерігати за ростом дерев та рослин.
# Ядро цінності
Основна цінність продукту - дати підлітку відчуття розуміння своїх емоцій та інструмент для самозаспокоєння. Користувач виконує прості завдання, що допомагають або вирішити складні ситуації в більш зрозумілому й простому вигляди, або дозволяють користувачу краще зрозуміти що саме він відчуває або відчував у тій чи іншій ситуації.
# Карта стейкхолдерів
Головною аудиторією є підлітки, які відчувають тривожність або бажають покращити свій ємоційний стан; психологи, які цікавляться подібним видом рішення проблем; а також розробник продукту. При використанні платформи користувач очікує покращення свого ємоційного стану, заспокоєння та підтримку.
# Definition of Success
Приблизно 10 користувачів написали відгук про покращення свого стану після використання пробної версії, кількість відвідувачів веб-платформи досягла хоча б 30.
